## mailinig-system

### Getting Started with MAIL APP

##Go to python directory 
`cd python`

## create virtualenv
`python3 -m virtualenv venv`

## activate virtualenv
`venv/bin/activate`

## Install packages 
`pip install django`
`pip install django-mptt`

## Run migration files
 `python manage.py migrate`

### start project: 
`python manage.py runserver`
